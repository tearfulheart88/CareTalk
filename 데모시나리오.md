### 2. 서버 상태 확인

```bash

curl http://127.0.0.1:9000/
# → {"server":"caretalk","version":"1.0.0","mock_mode":true,"tools":[...],"endpoint":"/mcp"}
```

### 3. Python 테스트 클라이언트 (권장)

curl은 한글 인코딩 이슈가 있으므로, Python으로 테스트합니다:

```python
import json, urllib.request

URL = "http://127.0.0.1:9000/mcp"

def call(method, params=None, id=1):
    payload = {"jsonrpc": "2.0", "method": method, "id": id}
    if params:
        payload["params"] = params
    req = urllib.request.Request(
        URL,
        data=json.dumps(payload, ensure_ascii=False).encode("utf-8"),
        headers={"Content-Type": "application/json"},
    )
    with urllib.request.urlopen(req) as r:
        return json.loads(r.read().decode("utf-8"))
```

## 데모 시나리오

### Step 0: Tool 목록 확인

```python
result = call("tools/list", id=0)
for t in result["result"]["tools"]:

**예상 결과**: 9개 Tool이 나열됩니다.
```
  - daily_checkin: 매일 안부 확인

  - family_report: 가족용 주간/일일 돌봄 리포트
  - daily_care_widget: 노인용 '오늘의 돌봄' Widget A
  - health_log: 건강 데이터 기록 및 추세 분석
  - reminiscence_chat: 추억 회상 정서 지원 대화
  - family_report_widget: 가족용 '주간 돌봄 리포트' Widget B

### Step 1: 아침 안부 발송 (daily_checkin — initiate)

**상황**: 매일 아침 8시, 돌봄톡이 순자 할머니에게 안부 메시지를 보냅니다.

```python

    "name": "daily_checkin",
    "arguments": {
        "user_id": "senior_001",
        "action": "initiate",

```
