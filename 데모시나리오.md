실행 후 콘솔에 다음 로그가 표시됩니다:

### 2. 서버 상태 확인

```bash
# 서버 정보 확인 (GET)
curl http://127.0.0.1:9000/
# → {"server":"caretalk","version":"1.0.0","mock_mode":true,"tools":[...],"endpoint":"/mcp"}
```

### 3. Python 테스트 클라이언트 (권장)

curl은 한글 인코딩 이슈가 있으므로, Python으로 테스트합니다:

```python
import json, urllib.request

URL = "http://127.0.0.1:9000/mcp"

def call(method, params=None, id=1):
    payload = {"jsonrpc": "2.0", "method": method, "id": id}
    if params:
        payload["params"] = params
    req = urllib.request.Request(
        URL,
        data=json.dumps(payload, ensure_ascii=False).encode("utf-8"),
        headers={"Content-Type": "application/json"},
    )
    with urllib.request.urlopen(req) as r:
        return json.loads(r.read().decode("utf-8"))
```

### Step 0.7: 가족 연결·예약 돌봄·휴대폰 활동 부재 전체 여정

**상황**: 순자님이 딸 한 명만 연결하고, 하루 세 번 안부 질문과 두 번의 가족 요약을 설정합니다.

```python
# 1) 어르신 동의 후 일회용 초대
invite = call_tool("care_circle", {
    "action": "create_invite",
    "requester_user_id": "senior_001",
    "senior_user_id": "senior_001",
    "nickname": "순자",
    "senior_consented": True
})

# 2) 지정 가족 가입. invite_code는 한 번만 사용 가능
call_tool("care_circle", {
    "action": "join",
    "requester_user_id": "family_001",
    "invite_code": invite["invite_code"],
    "nickname": "딸"
})

# 3) 예약과 활동 부재 기준 설정
call_tool("care_routine", {
    "action": "configure",
    "requester_user_id": "senior_001",
    "senior_user_id": "senior_001",
    "prompt_times": "09:00,14:00,20:00",
    "digest_times": "14:30,21:00",
    "inactivity_hours": 8,
    "inactivity_grace_minutes": 30,
    "senior_consented": True
})
```

휴대폰 어댑터가 `screen_unlock`과 `device_motion`의 발생 시각만 `record_activity`로 전달합니다.
운영 예약 작업은 `run_due`를 호출해 질문·요약·단계 알림을 중복 없이 대기열에 만듭니다.
가족 요약은 당일 전체 안부 질문과 응답, 식사·복약·활동·불편 신호를 합산하되 원문 답변은 전달하지 않습니다.

**예상 흐름**:

1. 화면 사용과 휴대폰 이동이 모두 8시간 멈춤
2. 어르신에게 `저는 괜찮아요`·`휴대폰을 두고 왔어요`·`도움이 필요해요` 확인
3. 30분 뒤에도 변화가 없을 때 권한 있는 딸에게만 활동 부재 안내
4. 딸이 `전화해볼게요` 또는 `방문 확인할게요`를 누르면 알림별 후속 행동 기록

> **안전 포인트**: 정확한 위치·화면 내용·원시 센서값을 저장하지 않고, 활동 부재를 응급으로 확정하지 않습니다. 가족의 일정 권한으로 활동 수집 범위나 철회된 동의를 바꿀 수도 없습니다. 실제 OS 신호 수집, 예약 실행, 카카오 발송에는 별도 동의형 앱·worker·승인 어댑터가 필요합니다.
